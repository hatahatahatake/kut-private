\subsection{MacOS 動作確認}
そのままでは HTML を閲覧することができないため，Safri の環境設定からプロキシ設定の変更を行う．\\
Safri の環境設定 → 詳細 → 詳細を変更 → 「Web プロキシ(HTTP)」と「保護された Web プロキシ(HTTPS)」のチェック項目を外す．\\
この作業により，閲覧が可能になり，自グループの HTML 文が表示される．
